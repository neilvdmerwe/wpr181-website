CSS Selectors

CSS selectors are patterns used to target specific HTML elements so you can apply styles to them. They tell the browser which elements a CSS rule should affect. There are several types of selectors, each serving a different purpose.

The type selector selects all elements of a given HTML tag.

p {
  color: blue;
}

This changes the text color of all <p> elements to blue.

The class selector selects elements with a specific class attribute. Classes start with a dot in CSS.

.highlight {
  background-color: yellow;
}

Any element with the attribute class="highlight" will have a yellow background.

The ID selector selects a single element with a specific id attribute. IDs start with a hash (#) in CSS.

#main-title {
  font-size: 24px;
}

Only the element with id="main-title" will get this style.

The descendant selector targets elements inside another element.

div p {
  color: green;
}

The element <p> is inside the <div> in this example. This makes all <p> elements inside a <div> green.

You can also use commas to apply the same style to multiple selectors. This works with elements, classes and IDs.

h1, #header-two, .cool-button {
  font-family: Arial, sans-serif;
}

CSS selectors also support wildcards.

* {
  margin: 0;
  padding: 0;
}

This removes all default margins and padding from every element.

Selectors are the foundation of CSS because they determine exactly where your styles will be applied. By combining them, you can target very specific parts of your page without affecting others.
