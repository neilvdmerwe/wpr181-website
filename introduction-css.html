Introduction to CSS

CSS stands for Cascading Style Sheets. It is the language used to control the appearance and layout of HTML elements on a web page. While HTML provides the structure of the page, CSS changes how that structure looks by controlling things like colors, fonts, spacing, and positioning.

CSS works by using rules that target HTML elements and apply styles to them. A CSS rule has two main parts: a selector and a set of properties. The selector chooses which elements to style, and the properties define what changes to make. Each property has a value that describes how it should look.

For example, if you want all paragraph text to be blue and slightly larger, you could write:

p {
  color: blue;
  font-size: 18px;
}
