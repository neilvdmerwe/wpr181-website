HTML Lists & Headings

Headings

Headings are used to create titles and subtitles in your content. HTML has six levels of headings, from <h1> to <h6>.

<h1> is the largest and most important heading, usually used for the main title.

<h6> is the smallest and least important.

Example:

<h1>Main Title</h1>
<h2>Section Title</h2>
<h3>Subsection Title</h3>

Lists

Lists are used to group related items together. There are two types of list elements in HTML. Both of these elements have the <li> tag inside of them, which stands for list item. The list item tag defines an item in the list, and can contain anything like text, paragraphs, links etc.

The first type of list is the unordered list. It displays a list of items using bullets.

<ul>
  <li>Apples</li>
  <li>Bananas</li>
  <li>Cherries</li>
</ul>

The page would look something like this:

  • Apples
  • Bananas
  • Cherries

The second type of list is the ordered list. This displays a list in a specific numbered order.

<ol>
  <li>First Step</li>
  <li>Second Step</li>
  <li>Third Step</li>
</ol>

It would look something like this:

  1. First Step
  2. Second Step
  3. Third Step
