HTML Document Layout

A normal HTML document has a specific structure that helps browsers understand and display the page correctly. It follows a top to-bottom layout, with certain required elements.

This is the basic structure of an HTML document: 

<!DOCTYPE html>
<html>
  <head>
    <title>My First Web Page</title>
  </head>
  <body>
    <h1>Welcome to My Website</h1>
    <p>This is my very first web page using HTML.</p>
  </body>
</html>

Let's break it down.

The first line declares the document type as HTML. This line is not required, but it's good practice to have it.

<!DOCTYPE html>

The second line starts with <html>. This element declares when the web page starts. You'll put all the elements you need to build the structure of the web page, in this element. Remember that this element has a closing tag </html> at the end of the document.

<html>

Like a human, it has a head, and then a body underneath, like this:

<head>
  ...
</head>
<body>
  ...
</body>

Inside the head, is the title of the web page. This is the title you see when you look at the tab of the web page.

<head>
  <title>My First Web Page</title>
</head>

Next is the body. The body contains all the elements that are actually displayed on the web page. Here you can use heading tags like <h1> or tags for text like <p>

<body>
  <h1>Welcome to My Website</h1>
  <p>This is my very first web page using HTML.</p>
</body>

And that's the basic structure of an HTML document. Take a good look at the example again.

<!DOCTYPE html>
<html>
  <head>
    <title>My First Web Page</title>
  </head>
  <body>
    <h1>Welcome to My Website</h1>
    <p>This is my very first web page using HTML.</p>
  </body>
</html>
