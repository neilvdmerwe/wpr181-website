HTML Elements

Elements

In HTML, elements are the basic building blocks that make up a web page.

An element usually has three parts: an opening tag, content, and a closing tag.

The opening tag tells the browser what kind of content is coming, and the closing tag tells it where that content ends.

For example, if you want to write a paragraph, you use the <p> element:

Here, <p> is the opening tag, I'm a paragraph! is the content, and </p> is the closing tag. The browser then knows to display that text as a paragraph.

Elements can also be nested inside each other. This is mostly used when you want specific contents of an element to display differently.

The <strong> element here is nested inside the <p> element, to make the words appear in bold.
