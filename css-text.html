CSS Text

CSS lets you control how text appears on your web page. There are several basic text styling features that you can use to make your content more readable and visually appealing.

Color

You can change the color of text using the color property. You can use color names, hexadecimal values, or RGB values. For example:

p {
  color: red;
}

This makes all paragraph text red.

Font Family

You can change the typeface of your text using the font-family property. You can provide multiple fonts as a fallback in case the first one is not available. For example:

p {
  font-family: Arial, sans-serif;
}

Font Size

You can adjust the size of your text with the font-size property. You can use pixels (px), ems (em), or percentages to define the size. For example:

p {
  font-size: 18px;
}

Font Weight

You can make text bolder or lighter using the font-weight property. Common values include normal, bold, or numeric values like 400 or 700. For example:

p {
  font-weight: bold;
}

Text Alignment

You can control how text is aligned within its container using the text-align property. Common values include left, center, right, and justify. For example:

p {
  text-align: center;
}
